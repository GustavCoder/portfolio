I'm having trouble with the flexbox layout in Module 4. When I try to make a responsive card layout, the cards are not wrapping correctly on mobile devices. Here's my code:

.card-container {
  display: flex;
  flex-wrap: wrap;
}
.card {
  width: 300px;
  margin: 10px;
}

Any help would be appreciated!

AJ
Alex Johnson (Instructor)
1 day ago

Hi John! The issue is that your cards have a fixed width of 300px. On small screens, this causes problems. Try using a percentage or flex-basis with max-width:

.card-container {
  display: flex;
  flex-wrap: wrap;
}
.card {
  flex: 1 0 300px; /* This means grow, don't shrink, basis 300px */
  max-width: 100%; /* Ensures it won't overflow on mobile */
  margin: 10px;
}

This should make your cards responsive. Let me know if it works!
